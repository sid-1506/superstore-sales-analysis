# 📊 Superstore Sales Analysis

## 📌 Overview
This project performs Exploratory Data Analysis (EDA) on a retail Superstore dataset containing 9,800+ sales records.  
The goal was to analyze revenue trends, identify top-performing categories and products, and extract business insights using Python.

---

## 🗂 Dataset Information
- Total Records: 9,800+
- Features: 18 columns
- Key Columns Used:
  - Category
  - Sub-Category
  - Region
  - Product Name
  - Sales

---

## 🔍 Key Analysis Performed

- ✅ Calculated total revenue (2.26M+)
- ✅ Category-wise sales analysis
- ✅ Sub-category-wise sales ranking
- ✅ Region-wise revenue distribution
- ✅ Top 5 revenue-generating products
- ✅ Visualized category sales using bar charts

---

## 📈 Sample Insight

- Technology is the highest revenue-generating category.
- East region contributed the highest sales.
- Canon imageCLASS 2200 Advanced Copier was among the top-selling products.
